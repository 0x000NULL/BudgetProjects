Help & Documentation

Getting Started

To get started with the Rental Car Claims system, follow these steps:

  1. Register a new account.
  2. Log in with your credentials.
  3. Navigate to the "Add Claim" page to submit a new claim.
  4. Use the "Email Templates" page to manage email templates.
  5. Use the "Dashboard" to view an overview of claims and recent activity.

Submitting a Claim

To submit a claim:

  1. Go to the "Add Claim" page.
  2. Fill in the required details including renter information and claim description.
  3. Upload any relevant images and files.
  4. Click "Submit Claim" to submit the claim.

Email Templates

To manage email templates:

  1. Go to the "Email Templates" page.
  2. Fill in the template name, subject, and body.
  3. Click "Save Template" to save the template.
  4. Select a template when viewing claims to send an email.

Dashboard

The dashboard provides an overview of the system including:

  • Total number of claims.
  • Claims by status.
  • Recent activity.
  • Charts and graphs to visualize data trends.
